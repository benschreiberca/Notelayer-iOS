# App Store Release Notes

**Version:** 1.5
**Build:** draft
**Timestamp:** 2026-02-09

## Highlights
- New `Insights` tab in root navigation (`Notes / To-Dos / Insights`)
- Historical task analytics across all available records, including completed tasks
- Rolling windows for tasks added/completed (`365/180/60/30/7`) plus all-time trend context
- Category analytics for added, completed, open, reminder, and calendar-export behavior
- Feature usage analytics with most/least rankings and feature gap analysis
- Time-of-day insights across task creation, completion, and app usage
- Progressive chart drill-down workflows powered by Swift Charts

## Why It Matters
This release turns Notelayer into a personal operations dashboard so users can understand and improve how they plan, complete, and manage tasks over time.

## Notes
- App usage analytics are local to each device in this release.
- App usage history begins when Insights telemetry is available on the device.
